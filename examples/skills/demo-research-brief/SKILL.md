# Demo Research Brief

Use this public-safe example skill to demonstrate how a reusable AI-agent skill can be stored in a registry, viewed in a UI, retrieved over MCP, and installed locally.

## Workflow

1. Identify the topic and audience.
2. Collect public source notes.
3. Summarize the key claims.
4. Flag uncertainty and missing evidence.
5. Produce a concise brief with next steps.
