# Temporary project memory

This directory is a short-term, public-safe project memory for Open SSR / Shared Skills Registry MCP while the launch work is moving quickly across multiple agents.

It is intentionally committed to the public repo for now so future agents can recover the current project state without relying on chat history.

Rules:

- Keep this directory free of secrets, tokens, private hostnames, private IPs, private customer/client references, and internal deployment paths.
- Record product decisions, commit checkpoints, open tasks, and public-safe implementation notes only.
- Do not treat this as permanent documentation.
- Delete this `docs/memory/` directory later after the public docs, issues, README, and/or release notes fully replace it.

Start with [`project-memory.md`](project-memory.md).
