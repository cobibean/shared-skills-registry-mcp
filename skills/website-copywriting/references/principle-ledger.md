# Website Copywriting Principle Ledger

| Principle | Sources | Website application | Skill rule |
|---|---|---|---|
| Conversion copy is salesmanship | Hopkins, Ogilvy | Every page gets a conversion job; every section earns its place. | If copy does not increase clarity, trust, desire, or action, cut it. |
| Persuasion must preserve trust | Cialdini | No fake scarcity, fake proof, guilt CTAs, or hidden pressure. | Use only naturally present persuasion. |
| Start from the reader’s job | Hopkins, Schwartz | Lead with user desire/anxiety, not company self-description. | Diagnose audience before writing. |
| Awareness controls the opening | Schwartz | Offer-first for hot traffic; education/problem-first for cold traffic. | Match lead type to awareness level. |
| Market sophistication controls believability | Schwartz | Stale markets need mechanism, stronger proof, or point-of-view. | If the claim sounds familiar, explain why/how this works. |
| One simple promise | Ogilvy | Hero/landing page carries one dominant promise. | Do not make five equal promises in the hero. |
| Big idea / reframing | Ogilvy, Schwartz | Use a central category/frame/mechanism, not feature pile. | Name the page’s big idea before drafting. |
| Specificity beats superlatives | Hopkins, Ogilvy | Replace vague claims with numbers, mechanisms, examples. | Cut unsupported “best/easy/powerful” language. |
| Proof near the claim | Cialdini, Hopkins | Pair promise with testimonials, data, demos, credentials, screenshots. | Mark `[proof needed]` rather than invent proof. |
| Peer-relevant social proof | Cialdini | Segment proof by role, industry, stage, or use case. | Logo soup is weak unless context is clear. |
| Real scarcity/loss | Cialdini | Use real capacity, deadline, or opportunity-cost framing. | Fake urgency is banned. |
| Reciprocity | Cialdini, Hopkins | Useful audits, demos, samples, templates, calculators. | Give real value before asking. |
| Consistency | Cialdini | Quizzes, self-selection, plan choice, saved preferences. | Commitments must be voluntary and user-owned. |
| Tell the full story | Hopkins, Schwartz | Longer pages for cold/high-anxiety buyers; structure for scanning. | Page length follows awareness gap. |
| Research finds the angle | Hopkins, Ogilvy | Mine user language, product facts, objections, competitors. | Do not wordsmith before the strategy is known. |
| Testing resolves uncertainty | Hopkins, Ogilvy | A/B test when traffic/stakes justify; otherwise use qualitative validation. | Do not settle serious copy debates by vibes alone. |
| Design supports the argument | Hopkins, Ogilvy | Visual hierarchy, scan path, proof placement, CTA affordance. | Pretty design cannot rescue weak claims. |
| Brand personality requires exclusion | Ogilvy, Schwartz | Choose audience, tone, point of view, non-goals. | Avoid copy that tries to be for everyone. |
