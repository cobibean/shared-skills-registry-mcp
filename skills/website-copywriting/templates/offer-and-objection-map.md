- Who is it for?

## Objection map

| Objection | Reader's hidden question | Evidence needed | Page location | Copy response |
|---|---|---|---|---|
| Fit | Will this work for me? | Use case / peer proof | Proof/use cases |  |
| Value | Is this worth it? | ROI / value metric | Pricing |  |
| Trust | Can I believe you? | Authority / testimonials / demo | Near claim |  |
| Risk | What if it fails? | Trial / guarantee / cancellation | CTA/pricing/FAQ |  |
| Effort | How hard is setup/switching? | Migration/onboarding proof | FAQ/comparison |  |
| Timing | Why act now? | Real deadline/capacity/opportunity cost | Offer/CTA |  |
| Security/privacy | Is this safe? | Security/compliance/data handling | Trust/FAQ/footer |  |
